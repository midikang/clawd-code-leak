# 深入 Claude Code - 交互式代码库课程

这是一个将 Claude Code 源代码转换成交互式学习课程的项目，使用中文编写。

## 📚 课程内容

本课程包含 6 个模块，带你从入门到精通 Claude Code 的架构设计：

1. **Claude Code 是什么？** - 产品介绍、技术栈、执行流程
2. **认识演员阵容** - 工具系统、命令系统、服务层
3. **工具如何工作？** - 工具生命周期、权限系统、BashTool 实例
4. **与外部世界对话** - API 集成、MCP、LSP
5. **聪明的技巧** - 并行预取、懒加载、特性开关、Agent Swarms
6. **全局架构** - 系统级视角、设计决策、实战应用

## 🎯 目标受众

"Vibe Coders" - 使用 AI 工具编程但缺乏传统计算机科学背景的开发者。

**学习目标：**
- ✅ 用正确的术语与 AI 交流
- ✅ 快速定位 bug 位置
- ✅ 要求 AI 应用性能优化模式
- ✅ 评估 AI 建议的安全性
- ✅ 理解 AI Agent 的工作方式

## 🚀 如何使用

### 在浏览器中打开

```bash
# 直接在浏览器中打开
open index.html

# 或使用 Python 启动本地服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

### 无需安装

课程是单个 HTML 文件，包含所有样式和交互脚本，**唯一的外部依赖是 Google Fonts CDN**。可以完全离线使用（除了字体）。

## ✨ 交互特性

- 📖 **代码翻译块** - 真实代码与中文解释并排显示
- 🎬 **动画可视化** - 数据流动画、组件通信动画
- 📊 **架构图** - 系统层次、组件关系
- ❓ **交互式测验** - 测试应用能力而非记忆
- 💡 **术语提示** - 悬浮显示技术术语的中文解释
- 🎨 **美观设计** - 温暖配色、清晰排版、流畅滚动

## 📁 目录结构

```
claude-code-course/
├── index.html          # 最终组装的课程（打开此文件）
├── styles.css          # 完整的样式系统

    ├── 03-tools.html

    ├── 05-patterns.html
    └── 06-architecture.html
```

## 🔧 技术实现

- **语言**: 纯 HTML + CSS + JavaScript（无框架）
- **字体**: Google Fonts (Bricolage Grotesque, DM Sans, JetBrains Mono)
- **代码高亮**: 内置语法高亮
- **滚动**: 原生 CSS scroll-snap
- **响应式**: 适配桌面和移动设备

## 📝 内容哲学

遵循 "codebase-to-course" 设计理念：

- **显示优于讲述** - 每屏至少 50% 是可视化内容
- **实战优于理论** - 测验题目是真实场景，不是定义背诵
- **应用优于记忆** - 教会如何使用知识，而非记住事实
- **中文友好** - 完全中文内容，精确的技术术语翻译

## 🎓 学习建议

1. **按顺序学习** - 模块之间有递进关系
2. **动手实践** - 打开 Claude Code 源码对照学习
3. **完成测验** - 测验题测试的是应用能力
4. **悬浮术语** - 不懂的术语鼠标悬浮即可查看解释
5. **重复浏览** - 第二遍会有新的发现
